# Sebaran Jumlah Suku di Indonesia

Halaman statis berisi 10 suku bangsa dengan populasi terbesar di Indonesia berdasarkan data Badan Pusat Statistik (BPS).

**Sumber data:** Badan Pusat Statistik

## Menjalankan secara lokal
Cukup buka `index.html` langsung di browser — tidak perlu server atau instalasi apa pun.
